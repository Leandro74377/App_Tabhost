## 🎯 FUNCIONALIDADES IMPLEMENTADAS

### 1️⃣ CUADRADO
```
Entrada: Lado (cm)
↓
Cálculos:
  • Área = lado²
  • Perímetro = 4 × lado
Ejemplo: lado=5
  → Área: 25.00 cm²
  → Perímetro: 20.00 cm
```

### 2️⃣ RECTÁNGULO
```
Entradas: Base y Altura (cm)
↓
Cálculos:
  • Área = base × altura
  • Perímetro = 2 × (base + altura)
Ejemplo: base=10, altura=5
  → Área: 50.00 cm²
  → Perímetro: 30.00 cm
```

### 3️⃣ TRIÁNGULO
```
Entradas: Base, Altura, Lado A, Lado B, Lado C (cm)
↓
Cálculos:
  • Área = (base × altura) / 2
  • Perímetro = A + B + C
Validación: Verifica desigualdad triangular
Ejemplo: base=6, altura=4, A=5, B=5, C=6
  → Área: 12.00 cm²
  → Perímetro: 16.00 cm
```

### 4️⃣ CÍRCULO
```
Entrada: Radio (cm)
↓
Cálculos:
  • Área = π × radio²
  • Perímetro = 2 × π × radio
Ejemplo: radio=5
  → Área: 78.54 cm²
  → Perímetro: 31.42 cm
```
